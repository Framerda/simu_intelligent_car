这是运行操作内容
1.下载项目.zip，解压用IntelliJ IDEA打开
2.在项目终端Terminal中输入 “mvn clean compile”指令
3.在目录中找到src/main/java/com/intelligentcar/IntelligentCarApplication.java 运行run IntelligentCarApplication.java
4.终端成功后会跳出 “
  智能小车Web控制平台启动成功！
  访问地址: http://localhost:8080
  WebSocket控制地址: ws://localhost:8080/ws/control
  WebSocket状态地址: ws://localhost:8080/ws/status
  REST API测试地址: http://localhost:8080/test
  ”
5.点击访问地址就可以查看前端了。
